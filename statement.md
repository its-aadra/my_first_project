## 1. Problem Statement
Many individuals, especially beginners and students, struggle with understanding how to manage their monthly income effectively. They lack clarity on budgeting, savings allocation, and suitable investment options. This project aims to provide a simple, interactive system that helps users calculate their financial distribution and receive basic investment recommendations.

## 2. Scope of the Project
- Validates user salary input.
- Applies the 50-30-20 budgeting framework.
- Generates categorized financial allocation (needs, wants, savings).
- Provides investment recommendations based on savings amount.
- Offers a repeatable, menu-driven user experience.
- Suitable for beginners and educational use.

## 3. Target Users
- Students and young professionals.
- Individuals with limited financial literacy.
- Users looking for a simple way to understand budgeting and investments.
- Anyone seeking quick, rule-based personal finance guidance.

## 4. High-Level Features
- Clean and intuitive menu-driven interface.
- Secure and validated salary input.
- Automatic 50-30-20 financial breakdown.
- Rule-based personalized investment suggestions.
- Error-free user experience using exception handling.
- Repeat cycle for multiple recommendations.
